# MovieRating_System
Movie rating system assignment for ITSE-2321. This program contains the use of user input, loops, arrays, logical operators, and switch cases. 
The function of this movie rating system is simple:
*A simple interactive rating system which takes user input and lays out results based on the given answers.*
>How it works:
- The user is greeted and is asked for their name.
- The user enters a name, which is then addressed throughout the rest of the program.
- The user is prompted to list their top 5 movies. This is done through the use of an array loop.
- The user is then prompted to give a number rating to each movie (1-10).
- The user is then asked to pick a favorite genre, by input of a number (1. action; 2. comedy; 3. horror; 4. drama; 5. sci-fi)
- the user is finally given a movie reccomendation based on the pick, and is dismissed with a warm thank you.
